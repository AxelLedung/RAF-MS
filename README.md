# RAF-MS™ - Robust Amazing Farm Management System

The RAF-MS - Robust Amazing Farm Management System is a old school assignment. Command Line Application for managing your animals and crops.

## Features

- Fully runs in the terminal/command-line interface (CLI)  
- Simple, beginner-friendly Java code  
- Can be extended or customized easily  

## Requirements

- Java 8 or higher installed  
- Terminal or command prompt to run the game  

## Installation

1. Clone the repository or download the source code.  
2. Open a terminal in the project directory.  
3. Compile the Java files:

```bash
javac *.java
